I’m trying to keep it simple stupid, so here are some steps to get started and deploying to gh-pages.

  1. install svelte
npm create svelte@latest myapp
cd myapp
npm install
npm run dev
  1. add a static preprocessor
npm i -D @sveltejs/adapter-static

and use it in svelte.config.js. 3. add gh-pages and a deploy step

npm i -D gh-pages
"deploy": "vite build && touch build/.nojekyll && gh-pages -d build -t true",
  1. wohoo, check your [username].github.io, if you have configured your repo/Settings/Pages to be deployed from gh-pages branch everything should be deployed 🎉
